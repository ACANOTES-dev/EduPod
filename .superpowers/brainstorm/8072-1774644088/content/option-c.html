All Three Options At a Glance

Navigation A B C

A: White top hub bar with text pills
B: Floating dark dock at the bottom with emoji icons
C: Dark cinematic top bar that morphs — expands a sub-strip inside modules

Home Page A B C

A: Uniform zone card grid + action chips
B: Bento grid with hero priority card
C: Action feed (left) + context panel (right). Feed-driven, not card-driven.

Module Depth A B C

A: Sidebar sub-nav for complex modules
B: Horizontal tabs (no sidebar anywhere)
C: Morphing bar — sub-tabs live inside the dark bar itself

Content Width A B C

A: Full-width on home, narrower in sidebar modules
B: Full-width everywhere
C: Full-width everywhere (sub-nav is in the bar, not the content)

Personality A B C

A: Clean, bright, accessible. Linear + ClassDojo.
B: App-like, spatial, playful. Apple + Arc.
C: Cinematic, editorial, confident. Notion + Superhuman.

Mobile Story A B C

A: Hub bar → bottom tabs
B: Dock already at bottom — near-zero adaptation
C: Morph bar → hamburger + bottom tabs (needs more work)

The question isn't which is "best" — it's which personality matches EduPod's soul. A is the safest, most universally appealing. B is the boldest and most app-like. C is the most sophisticated and editorial. All three use the same unified-layers architecture. The difference is purely in how they express it.
