# Health Governance

The canonical repo home for health-governance rules is `Governance/`.

Before closing health-related work, open these files as needed:

1. `Governance/README.md` for the control-plane index
2. `Governance/recovery-backlog.md` for tracked risks, owners, due dates, and retirement evidence
3. `Governance/governance-policy.md` for written tradeoffs, retirement rules, and critical-workflow completeness
4. `Governance/review-cadence.md` for the weekly health review rhythm
5. `Governance/scorecard-metrics.md` and `Governance/monthly-scorecards/` for the KPI registry and published scorecards
6. `Governance/re-audit-checkpoints.md` for Wave 1, Wave 3, and final independent re-audit gates

Treat `.claude/rules/health-governance.md` as the loader and `Governance/` as the source of truth. If the governing reality changes, update the file in `Governance/` that owns that rule in the same change.
