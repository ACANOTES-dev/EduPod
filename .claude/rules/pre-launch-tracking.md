# Pre-Launch Item Tracking

## When the user defers something to pre-launch

If the user says something like "that's a pre-launch item", "we'll handle that before launch", "defer to launch prep", or similar:

1. Add the item to `Manuals/PRE-LAUNCH-CHECKLIST.md` → Part 5 (Deferred Items)
2. Include: what the item is, where it was flagged, today's date, and any relevant notes

## When the user indicates they're preparing for launch

If the user says "we're preparing for launch", "let's get ready to go live", "time to launch", or similar:

1. Read `Manuals/PRE-LAUNCH-CHECKLIST.md` in full
2. Review every item — flag anything that hasn't been verified
3. For Part 2 (known gaps): verify against current code whether each gap has been fixed
4. For Part 5 (deferred items): ensure each item has been addressed
5. Present a summary of what's done vs what's outstanding
