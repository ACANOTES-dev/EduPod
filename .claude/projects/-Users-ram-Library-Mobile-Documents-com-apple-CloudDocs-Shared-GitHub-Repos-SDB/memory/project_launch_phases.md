The project roadmap has Phases 1-4 as pre-launch:
- Phase 1 (P0-P9): Core feature build — ~90% complete as of March 2025
- Phase 2: Refinements, QA fixes, tenant feedback iterations (current work)
- Phase 3: Codebase refactor to add 8 languages (French, Spanish, Irish, Polish, Romanian, German, Chinese, Japanese) on top of existing English/Arabic
- Phase 4: Pre-launch testing and preparation

**Why:** The product is not live yet. Two confirmed tenants pending onboarding. All current work is pre-launch preparation.

**How to apply:** Do not treat any feature as "shipped" or "in production" — everything is still in pre-launch mode. Optimise for correctness and completeness, not hotfix speed.
