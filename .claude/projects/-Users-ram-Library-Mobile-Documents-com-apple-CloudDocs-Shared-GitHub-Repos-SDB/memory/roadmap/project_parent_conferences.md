**Priority:** Nice-to-have. Not a blocker but a quality-of-life feature that impresses during demos.

**What it is:**
An online booking system where parents self-schedule time slots with their child's teachers for parent-teacher meetings. Replaces the paper-based or email-based scheduling that most schools still use.

**What EduPod should build:**
- Conference creation: admin sets date, time range, slot duration (e.g., 10 minutes), buffer between slots
- Teacher availability: teachers mark which slots they're available
- Parent booking: parents book slots with each of their child's teachers via the parent portal
- Conflict prevention: can't double-book a teacher or a parent in the same slot
- Automated reminders: email/push notification before the conference
- Schedule view: teachers see their full evening schedule; parents see their booking confirmations
- Walk-in support: admin can manually book for parents who don't use the portal
- Video call integration (optional): link to Teams/Zoom for remote conferences

**Competitor status:**
- Compass: Parent Conferences module included
- VSware: not offered
- Most schools use paper sign-up sheets or manual email coordination

**How to apply:** Build after higher-priority items. This is a low-effort, high-visibility feature that demonstrates the "all-in-one" value proposition. Schools that currently coordinate parent evenings manually will immediately see the time savings.
