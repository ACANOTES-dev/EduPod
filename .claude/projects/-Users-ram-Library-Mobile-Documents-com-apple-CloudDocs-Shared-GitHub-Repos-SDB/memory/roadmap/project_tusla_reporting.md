**Priority:** Non-negotiable. Legal requirement for all Irish schools.

**What it is:**
Under the Education (Welfare) Act 2000, schools must report to Tusla (Child and Family Agency):
1. **Student Absence Reports (SAR):** Report all students absent 20+ days, submitted twice per year via Tusla Portal (portal.tusla.ie). Covers children aged 6-16.
2. **Annual Attendance Report (AAR):** Total days lost through absence, submitted within 6 weeks of school year end.

**What EduPod must do:**
- Track cumulative absences per student against the 20-day threshold
- Generate Tusla-compliant export files (SAR format) for portal upload
- Generate Annual Attendance Report data
- Support Tusla attendance codes (illness, unexplained, suspended, etc.)
- Dashboard/alert when students approach the 20-day threshold
- TESS (Tusla Education Support Service) dashboard support

**Competitor status:**
- Compass: automatic Tusla export, attendance codes aligned with Tusla requirements
- VSware: full Tusla reporting with NEWB/Tusla codes, absence threshold monitoring
- Both have this battle-tested

**Effort estimate:** Low-Medium — EduPod already has sophisticated attendance tracking with pattern detection. The 20-day threshold detection is similar to existing pattern alerts. Main work is mapping to Tusla's specific code set and export format.

**How to apply:** Must be built before approaching any Irish school. The attendance module already detects patterns — extending it to Tusla-specific thresholds and export format is a natural evolution.
