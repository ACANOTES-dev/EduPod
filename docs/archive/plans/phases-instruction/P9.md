## TESTING REQUIREMENTS

### RLS Leakage Test Suite

- Automated tests that attempt cross-tenant data access via every API endpoint
- Tests must cover: direct queries, search results, notification dispatch, queue payloads
- Every tenant-scoped table must have at least one leakage test

### Visual Regression Test Suite Hardening

- Expand Playwright visual comparisons beyond Phase 0 foundation to cover all module screens in both locales, dark mode variants, and mobile breakpoints
- Target: 100% coverage of record hub pages, all dashboard views, all form wizards, and the print preview modal

### PDF Snapshot Test Suite

- Snapshot tests for every Puppeteer template in both locales (en, ar)
- Templates: report cards, transcripts, receipts, invoices, payslips
- CI must fail if rendered output changes unexpectedly

### RTL Regression Test Suite

- Visual regression tests for all school-facing views in Arabic locale
- Verify Tailwind logical utilities are used (no physical left/right)
- Verify LTR enforcement on emails, URLs, phone numbers, numeric inputs

### Integration Test Suites for Critical Workflows

- 5 critical workflow suites, each running against real PostgreSQL with RLS enabled (not mocked):
  1. **Admissions conversion**: multi-entity creation in single transaction
  2. **Refund LIFO reversal**: allocation deletion/reduction + invoice re-derivation
  3. **Household merge**: financial record reassignment + concurrency guard
  4. **Payroll finalisation**: entry freeze + payslip generation + approval flow
  5. **Payment allocation**: FIFO suggestion + over-allocation prevention + invoice status derivation

### Load/Performance Testing

- Simulate 100 concurrent users across 5 tenants
- Test: login, search, attendance marking, invoice generation, payroll finalisation
- Identify bottlenecks in RLS context switching, Meilisearch sync, PDF rendering

### Schedule Buffer Note

Phases 0-9 (including Phase 4b auto-scheduling) sum to 35 weeks. A 5-week contingency buffer (14%) is recommended, bringing the realistic timeline to 40 weeks (~10 months). Finance (Phase 6), Payroll (Phase 6b), and Auto-Scheduling (Phase 4b) carry the highest schedule risk.
