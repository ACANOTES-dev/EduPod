**Priority:** Vision / post-launch. High impact but large scope. The one feature that would make students use the platform daily.

**What it is:**
A digital replacement for the physical homework journal/diary that every Irish post-primary student carries. Teachers set homework digitally per class, students see it in their portal, parents see it in the daily digest, and completion is tracked. This is the feature that creates daily engagement from ALL three user types — teachers, students, and parents.

**What EduPod should build:**

- **Teacher homework setting:**
  - Quick entry: select class, type description, set due date (under 30 seconds to set)
  - Homework types: written, reading, research, revision, project work, online activity
  - Attach files/links (worksheets, resources, URLs)
  - Recurring homework (e.g., "read 20 minutes nightly")
  - Copy from previous year/week
  - Link to gradebook assessment (optional — homework feeds into grade)
- **Student view:**
  - Today's homework across all subjects
  - This week's overview
  - Overdue items highlighted
  - "Mark as done" checkbox (self-reported)
  - Upcoming due dates calendar
- **Parent visibility:**
  - See homework set for their child per subject
  - See what's overdue
  - Included in the smart daily digest
  - Weekly summary option
- **Teacher tracking:**
  - Class completion rates (e.g., "22/28 students marked maths homework as done")
  - Identify students consistently not completing homework
  - Homework load analysis (are students getting too much across all subjects?)
- **School diary features:**
  - Daily notes (student-to-self, not visible to others)
  - School events and deadlines in the diary view
  - Parent-teacher communication notes (replaces the "sign the journal" workflow)
- **Analytics:**
  - Homework completion trends per student, per class, per subject
  - Cross-subject homework load analysis (school-wide — are Wednesdays overloaded?)
  - Correlation with academic performance (do students who complete homework consistently score higher?)

**Competitor status:**

- No school MIS in Ireland offers this
- Standalone homework apps exist (Show My Homework / Satchel One in the UK) but aren't integrated with MIS
- Ireland still uses paper journals almost universally

**Why it's revolutionary:**

- The homework journal is the ONE thing every student, teacher, and parent interacts with DAILY
- Digitising it creates daily engagement that no other feature can match
- Parents of students with additional needs particularly benefit — they can see exactly what's expected
- The data feeds into predictive early warning (homework completion is a leading indicator of academic risk)
- Cross-subject load analysis is something no school currently has visibility into

**Dependencies:** Benefits from mobile app (students check homework on phones). Works without it but engagement is much higher with mobile.

**Effort estimate:** Medium-High — new data models, student-facing UI (which is mostly parent-portal-like), teacher quick-entry UI, analytics. The patterns exist but the student engagement layer is new territory.

**How to apply:** This is a post-launch feature. Build after core MIS is established with pilot schools and after the mobile app exists. When pitching, mention it as "on our roadmap" — it's the kind of feature that generates excitement even before it's built. It also differentiates clearly from every competitor: "we're replacing the school journal."
